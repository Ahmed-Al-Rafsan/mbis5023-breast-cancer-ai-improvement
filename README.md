# MBIS5023 Breast Cancer CNN Improvement

This repository supports the A2 open AI-assisted process report for MBIS5023 Machine Learning for Data Analytics.

## Project aim

The project improves an A1 breast cancer classification artefact by comparing a basic one-dimensional CNN with a more reliable pipeline. The improved workflow adds standardisation, a validation split, class weighting, dropout, batch normalisation, early stopping, reproducibility controls and expanded evaluation.

## Dataset

Breast Cancer Wisconsin (Diagnostic), supplied through the unit portfolio materials.

- `notebooks/A2_AI_Assisted_Breast_Cancer_CNN_Improvement.ipynb`

## Reproduction

1. Install packages from `requirements.txt`.
2. Open the notebook.
3. Run all cells from top to bottom.
4. Review the metric comparison and confusion matrices.

The model is an academic demonstration and is not intended for clinical diagnosis.
